##	使用

#### linux

```shell
wget https://github.com/Asuka999/szpt-pro/releases/download/V1.0/szptPro-linux-386.tar
tar -xvf ./szptPro-linux-386.tar
cd ./szptPro
# edit user.json
nohop ./szptPro & 
```
